# net-tools

net-tools is actually a collection of a number of different commands, listed here:

https://net-tools.sourceforge.io/

Source Code:

https://sourceforge.net/projects/net-tools/

The tools include:

* arp(8), 
* hostname(1), 
* ifconfig(8), 
* ipmaddr, 
* iptunnel, 
* mii-tool(8), 
* nameif(8), 
* netstat(8), 
* plipconfig(8), 
* rarp(8), 
* route(8)
* slattach(8).
